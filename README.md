# 受講生管理システム

受講生の情報を一元管理し、コースの状態や申し込み経路を可視化するシステムです。

## 概要

AIを活用した個人開発のサンプルプロジェクトとして構築。単なるモックではなく、実務でも通用する本格的なシステムを目指しています。

### 主な機能

- 受講生情報の管理（登録、編集、状態管理）
- コース情報のマスタ管理
- 決済状態の管理（入金確認）
- 申し込みフォーム・ヒアリングフォーム
- ロールベースのアクセス制御
- 外部システム連携（チャット、決済）

## 開発ルール

### ブランチ運用

- **メインブランチ（main）での直接作業は禁止**
- **ブランチ作成時は必ずIssueとセットで作成**
- ブランチ命名: `{タイプ}/{Issue番号}-{説明}`
  - 例: `feature/123-add-student-list`

### コミットメッセージ

Conventional Commits形式を使用：

```
{タイプ}: {説明} (#{Issue番号})
```

例: `feat: 受講生一覧APIを実装 (#123)`

詳細は [CLAUDE.md](CLAUDE.md) を参照してください。
